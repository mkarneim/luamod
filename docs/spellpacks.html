    Spell Packs

    A spell pack is a convenient way of sharing spells between wizards and server owners. Technically it’s a Wizards of Lua add-on that contains one or more Lua files bundled into a Jar file.

    Server owners can add a spell pack to their server as any “normal” Forge-based Minecraft modification by dropping it into the mods folder. When placed into the server’s mods folder (next to the WoL JAR file) the spell pack will be added to the Lua search path.

    Spell packs can be created by using the /wol pack export command.
