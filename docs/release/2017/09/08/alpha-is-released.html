    Alpha 1.2.1 is Released

    The new 1.2.1-alpha release contains bugfixes.

    • Fixed #50 - Placing a door using spell.block is not working
    • Fixed #51 - luaTicksLimit is not stored correctly into config file when changed with /wol command
    • Fixed #52 - Add automatic version checking

    Especially fix #50 is important since it improves the performance when a lot of blocks are changed in a short period of time.
