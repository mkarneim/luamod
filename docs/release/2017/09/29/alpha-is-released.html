    Alpha 1.5.0 is Released

    The 1.5.0-alpha is available. It brings a ton of new entity properties and adds support for relative movement to Entity.move().

    • Fixed #35 - Spells should read and write more entity properties: rotationYaw, rotationPitch, eyeHeight, lookVec, tags, facing, motion. We also have a new Player property: team.
    • Fixed #58 - Entity.putNbt() should support setting tags
    • Fixed #61 - Rename “Runtime” module to “Time” and replace most function with properties
    • Fixed #62 - A new spell’s facing should be the same as the wizard’s facing
    • Fixed #63 - Entity.move() should accept relative directions (forward, back, left, right)
    • Fixed #64 - Replacing a block with modified nbt not working

    Please note that this version is not backwards compatible since the former Runtime module has been renamed to Time.
