    The Art of Spell Crafting

    Features

    The Wizards of Lua modification adds the /lua command to Minecraft, which gives you the power to cast magical spells in Minecraft.

    With a spell you can, for example, do the following:

    • Create magical items.
    • Create mini games.
    • Create procedural structures.

    To do all this, you just need to become a Wizard of Lua. That means, getting familiar with Lua, the language of magic, and studying the magic words documented in the secretive spell book library.

    As a wizard, you will know how to execute the following great magical powers:

    • Creating spells with the built-in Lua editor.
    • Using all Lua language features, like variables, loops, conditional bocks, and functions.
    • Running multiple spells simultaneously.
    • Breaking spells.
    • Accessing and changing any block in the Minecraft world, including data values and NBT values.
    • Querying and manipulating players, creatures, and items.
    • Receiving in-game events like left-click-block, swing-arm, and many others.
    • Firing and handling custom events.
    • Executing any Minecraft command, even those added by other mods.
    • Accessing self-made Lua modules from the serverâ€™s file system.
